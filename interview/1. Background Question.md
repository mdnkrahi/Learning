# Question 4: Describe one AI project you're most proud of.

**Answer:**

"I'm particularly proud of building an end-to-end Retrieval-Augmented Generation (RAG) application for enterprise document search.

The project involved:

* Extracting data from PDFs and other business documents.
* Splitting and preprocessing documents.
* Generating embeddings using transformer models.
* Storing embeddings in a vector database.
* Retrieving relevant context based on user queries.
* Using a Large Language Model to generate accurate responses.
* Deploying the complete solution on AWS using Docker and Kubernetes.

I also implemented monitoring, logging, and automated deployment pipelines, making the solution production-ready.

The biggest challenge was improving response accuracy and reducing hallucinations. I solved this by optimizing chunk sizes, improving retrieval strategies, and refining prompt engineering.

This project strengthened my expertise in LLMs, vector databases, AWS, MLOps, and production AI deployment."
